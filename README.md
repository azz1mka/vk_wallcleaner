# VK Wall Cleaner

Скрипт для очистки стены ВКонтакте, не требующий установки программ. Удаляет большое количество записей на стене без вашего участия.
 
## Использование

1. Скопируйте содержимое **delete_wall.js**
2. Откройте свою страницу ВКонтакте
3. Нажмите **F12 → Console**
4. Вставьте содержимое `delete_wall.js` (консоль может блокировать вставку, для этого введите allow pasting) 
5. Вставьте свои данные:

```javascript
const token = "здесь access_token";
const owner_id = "здесь user_id";
```

6. Нажмите **Enter**


После этого начнётся автоматическое удаление постов.
---
## Получение Owner ID
```
https://vk.com/idXXXXXXXX
```
Скопируйте значение после `id`

В случае если ID менялся на кастомный:

Откройте свою страницу ВКонтакте
1. Нажмите **F12 → Console**
2. Введите `vk.id`
3. Нажмите **Enter**
4. Скопируйте цифры
---
## Получение Access Token

Откройте в браузере ссылку:

```
https://oauth.vk.com/authorize?client_id=6287487&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=wall,offline&response_type=token&v=5.131
```
После авторизации откроется страница с адресом вида:

```
https://oauth.vk.com/blank.html#access_token=vk1.a.XXXXXXXXXXXX&expires_in=0&user_id=123456
```

Скопируйте значение после `access_token=`.

## Ограничения API

API ВКонтакте имеет ограничения на количество запросов.
В скрипте используется задержка между запросами, чтобы избежать ошибки `Too many requests`.
